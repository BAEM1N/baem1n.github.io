iTerm2 설치

링크를 클릭하여 iTerm2를 다운로드 받습니다.

iTerm2는 Terminal을 대체할 수 있는 프로그램입니다.

다운로드 받은 파일을 해제하여 생성된 iTerm을 응용프로그램으로 이동 후 프로그램을 복사하여 다음과 같이 iTerm (Rosttea)로 이름을 변경합니다.

iTerm (Rosttea)를 선택 후 보조 클릭을 통해 정보 가져오기 메뉴를 클릭후 Rosetta를 사용하여 열기 옵션을 체크합니다.
