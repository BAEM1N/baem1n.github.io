블로그 배포 전에 매번 체크하는 항목들.

## 검색 노출

- canonical URL 확인
- sitemap 생성 및 제출
- robots.txt에 sitemap 경로 선언
- 구조화 데이터(JSON-LD) 기본 적용

## 분석

- GA4 측정 ID 연동
- 중요한 클릭 이벤트 추적 계획
- Search Console 연동

## 운영

- GitHub Actions 배포 성공 여부 확인
- 배포 후 Lighthouse 핵심 지표 확인
- 첫 글 색인 요청

작게 시작해도, 체크리스트를 꾸준히 지키면 누적 효과가 꽤 크다.
