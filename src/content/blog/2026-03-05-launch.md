오늘 개인 개발 블로그를 오픈했다.

핵심 목표는 다음 4가지다.

1. 글 작성이 빠를 것
2. SEO 기본기가 탄탄할 것
3. GA4로 트래픽을 확인할 수 있을 것
4. AI 검색 크롤러가 읽기 쉬운 구조일 것

초기 스택은 **Astro + GitHub Pages + GitHub Actions**로 선택했다.

앞으로는 실제 운영하면서 얻은 트러블슈팅과 자동화 노트를 중심으로 기록할 예정이다.
